# How to setup project and run locally

### Clone the repository 

```
git clone https://github.com/gcodezz/Linkop-functions.git

```

### Install all dependencies

Using npm

```
npm install
```

### Start watching the file and changes

Using npm

```
npm start --reset-cache
```
